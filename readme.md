### Prerequistes

You will need :
 - [java8](https://jdk8.java.net/download.html)
 - [maven](http://maven.apache.org)
 - [bower](http://bower.io)

### Build

    $ bower install
    $ mvn package

### Deploy locally

    $ mvn tomcat7:run

And [try it locally](http://localhost:8080).
